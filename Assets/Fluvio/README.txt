Thank you for downloading Fluvio!
To install Fluvio, go to Window > Fluvio > Install Fluvio. To get started, go to Window > Fluvio > Welcome.

Once installed, the reference manual for this version can be found by going to Window > Fluvio > Reference Manual. This manual can also be downloaded locally by going to the Fluvio preference pane in the Unity preferences.

Up-to-date documentation is available at:
https://docs.getfluv.io

Product support can be found at:
https://support.getfluv.io

Example scenes and additional Fluvio packages can be found at:
https://getfluv.io/explore
